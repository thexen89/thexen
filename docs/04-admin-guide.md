# 관리자 기능 가이드

## 접속

1. `https://사이트주소/admin/login` 접속
2. 관리자 비밀번호 입력
3. 24시간 세션 유지

## 제품 관리

### 제품 추가
1. 우측 상단 "새 제품 추가" 버튼 클릭
2. 제품명, 고객사, 설명 입력
3. 이미지 업로드 (여러 장 가능)
4. 썸네일 이미지 선택 (그리드에 표시될 이미지)
5. 동영상 URL 입력 (YouTube/Vimeo, 선택사항)
6. 저장

### 제품 수정
- 제품 카드의 편집 버튼 클릭
- 정보 수정 후 저장

### 제품 삭제
- 제품 카드의 삭제 버튼 클릭
- 확인 후 삭제 (Cloudinary 이미지도 함께 삭제)

### 제품 순서 변경
- **PC**: 그리드에서 제품을 드래그 앤 드롭
- **모바일**: 제품을 길게 누르면 순서변경 모드 진입 → 드래그

## 사이트 설정

### 랜딩 페이지
| 설정 | 설명 |
|------|------|
| 로고 이미지 | 랜딩 화면 중앙 로고 (미설정 시 "THEXEN" 텍스트) |
| 배경 이미지 | 랜딩 화면 배경 |
| 배경 타입 | `tile` (반복 패턴) 또는 `cover` (전체 덮기) |
| Enter 이미지 | 하단 진입 버튼 아이콘 |

### 그리드 화면
| 설정 | 설명 |
|------|------|
| 배경 색상 | 포트폴리오 그리드 배경색 (HEX 코드) |
| 헤더 로고 | 좌측 상단 로고 (클릭 시 회사 소개 모달) |
| 외부 링크 | 우측 아이콘 + URL (인스타그램, 카카오 등) |

### 패널 위치 조정
- 좌측 패널 (About/로고): X, Y 좌표 드래그로 위치 조정
- 우측 패널 (외부 링크): X, Y 좌표 드래그로 위치 조정

### 시즌 이펙트
| 이펙트 | 설명 |
|--------|------|
| `snow` | 눈 내리기 |
| `cherry` | 벚꽃 흩날리기 |
| `leaves` | 낙엽 떨어지기 |
| `fireworks` | 불꽃놀이 |

### 회사 소개
- 회사 소개 이미지 (여러 장, 슬라이드)
- 회사 소개 텍스트

## 주의사항

- 제품/설정 변경 후 **최대 60초** 후 반영 (CDN 캐시)
- 이미지는 Cloudinary에 저장되므로 용량 제한 확인
- 업로드 파일 최대 크기: 10MB
