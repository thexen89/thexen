## 수정 계획 (우선순위순)

### Phase 1: 즉시 수정 가능 (난이도 하)

#### 1-1. 그림자 방향 조정 (5시 방향)
- **파일**: `src/components/HexGrid.tsx`
- **현재**: 전방향 균일 그림자
- **변경**: `boxShadow`를 `4px 6px 20px rgba(0,0,0,0.6)` 형태로 변경 (x: 양수, y: 양수 = 오른쪽 아래)

#### 1-2. 상세페이지 좌우 이동 아이콘 추가
- **파일**: `src/app/product/[id]/page.tsx` 또는 이미지 슬라이더 컴포넌트
- **작업**: 슬라이더 좌우에 화살표 버튼 추가
- **디자인**: `<` `>` 또는 chevron 아이콘

#### 1-3. 키보드 방향키 지원
- **파일**: 상세페이지 컴포넌트
- **작업**: `useEffect`로 `keydown` 이벤트 리스너 추가
- **키**: ArrowLeft, ArrowRight

#### 1-4. 뒤로가기 버튼 추가
- **파일**: 상세페이지
- **작업**: 상단에 뒤로가기 버튼 추가 (포트폴리오 페이지로 이동)

### Phase 2: 중간 난이도

#### 2-1. 10초 idle 타이머 (메인페이지 자동 이동)
- **파일**: 상세페이지
- **작업**:
  - `mousemove`, `keydown`, `click`, `scroll` 이벤트 감지
  - 10초간 이벤트 없으면 메인페이지로 리다이렉트
  - 사용자 경고 토스트 표시 (선택)

#### 2-2. 메인페이지 카테고리 메뉴 추가
- **파일**: `src/app/page.tsx`
- **작업**: 헤더에 디자인/판촉분야/온라인스토어/더보기 메뉴 추가
- **참고**: 현재 Shop 버튼이 있는 위치 활용 가능

#### 2-3. 비디오 슬라이더 지원
- **파일**: 상세페이지 슬라이더
- **작업**:
  - 상품 데이터에 `videoUrl` 필드 추가
  - YouTube/Vimeo URL 감지 및 embed iframe 렌더링
  - 썸네일 클릭 시 비디오 재생

### Phase 3: 고난이도 (추가 협의 필요)

#### 3-1. 아이콘 모양 선택 (원/사각형/육각형)
- **파일**:
  - `src/components/HexGrid.tsx` - 모양별 렌더링
  - `src/app/admin/page.tsx` - 설정 UI
  - `prisma/schema.prisma` - `gridShape` 필드 추가
- **작업**:
  - 사각형: `rounded-none` 또는 `rounded-lg`
  - 육각형: CSS clip-path 또는 SVG 마스크

#### 3-2. 그리드 영역별 디자인 커스터마이징
- **협의 필요 사항**:
  - "블록" 영역이 정확히 어디인지? (좌측/우측/상단/중앙?)
  - 색상만 변경? 이미지 배경도 가능?
  - 각 영역의 크기 조절 가능 여부?

#### 3-3. 사각형 그리드 레이아웃 옵션
- **협의 필요 사항**:
  - 100칸 형태의 정확한 의미 (10x10? 가로 스크롤?)
  - 원형 그리드와 전환 가능해야 하는지?

## 협의가 필요한 항목

1. **원 정렬 vs 벌집 구조**: 현재 벌집(hexagon) 배열을 유지할지, 정렬된 그리드로 변경할지?
2. **블록 디자인 커스터마이징**: 정확히 어떤 영역을 색상/이미지로 채울 수 있게 할지?
3. **100칸 형태**: 사각형 그리드의 정확한 레이아웃 형태?
4. **백보드 움직임**: 배경이 어떻게 움직여야 하는지?
